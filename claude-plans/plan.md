# MP3 Player Perfection Plan

## Core Focus: Local Music Player Excellence

### 1. Library & Playlist Management
- [ ] **Expanded playlist view**
  - [ ] Import/display local .m3u/.m3u8 playlists
  - [ ] Tree view for Artist → Album → Track hierarchy
  - [ ] Genre-based organization
  - [ ] Search functionality across library
  - [ ] Smart playlists (recently added, most played, etc.)

- [ ] **Playlist editing**
  - [ ] Create/edit/delete custom playlists
  - [ ] Drag & drop to reorder tracks
  - [ ] Drag & drop to add tracks to playlists
  - [ ] Export playlists to .m3u format

### 2. System Integration
- [ ] **Mac keyboard media hotkeys**
  - [ ] Play/pause (F8 or media key)
  - [ ] Previous track (F7)
  - [ ] Next track (F9)
  - [ ] Volume up/down (F10/F11/F12)

- [ ] **macOS media controls**
  - [ ] Now Playing widget in Control Center
  - [ ] Lock screen media controls
  - [ ] Dock icon controls (right-click menu)
  - [ ] Menu bar mini controls (optional)

### 3. Audio Experience
- [ ] **Playback features**
  - [ ] Gapless playback for albums
  - [ ] Crossfade between tracks (configurable)
  - [ ] Shuffle modes (tracks, albums)
  - [ ] Repeat modes (off, one, all)
  - [ ] Seek/scrub with keyboard (arrow keys)

- [ ] **Audio quality**
  - [ ] Support for FLAC, AAC, MP3, M4A formats
  - [ ] Basic equalizer (preset + custom)
  - [ ] Volume normalization option

### 4. User Interface
- [ ] **Player views**
  - [ ] Mini player mode (compact)
  - [ ] Full player with album art
  - [ ] Queue/up next view
  - [ ] Lyrics display (if embedded)

- [ ] **Navigation & shortcuts**
  - [ ] Keyboard shortcuts for all actions
  - [ ] Quick search (Cmd+F)
  - [ ] Recently played section
  - [ ] Favorites/starred tracks

### 5. Metadata & Organization
- [ ] **Track information**
  - [ ] Display: Title, Artist, Album, Duration, Bitrate
  - [ ] Album art extraction and display
  - [ ] Year, Genre, Track numbers
  - [ ] File path and format info

- [ ] **Library scanning**
  - [ ] Auto-scan music folders on startup
  - [ ] Manual rescan option
  - [ ] Watch for file system changes
  - [ ] Handle moved/deleted files gracefully

### 6. Import & Export
- [ ] **File management**
  - [ ] Drag & drop music files to add to library
  - [ ] Import from iTunes/Music.app library
  - [ ] Bulk import from folders
  - [ ] Duplicate detection

### 7. Performance & Polish
- [ ] **Optimization**
  - [ ] Fast library loading
  - [ ] Smooth scrolling in large lists
  - [ ] Efficient memory usage for audio
  - [ ] Background audio when window closed

- [ ] **Error handling**
  - [ ] Graceful handling of corrupted files
  - [ ] Network drive disconnection handling
  - [ ] Missing file recovery

## Phase 1 Priorities (MVP)
1. Mac media hotkeys integration
2. Expanded playlist view with .m3u support
3. Artist/Album tree view
4. Basic playback controls improvement
5. Library scanning and organization

## Phase 2 Enhancements
1. Advanced audio features (crossfade, gapless)
2. Mini player mode
3. Keyboard shortcuts
4. Equalizer
5. System media controls integration

## Phase 3 Polish
1. Import/export features
2. Advanced playlist management
3. Performance optimizations
4. UI/UX refinements
